# Nand2Tetris

## Usage

### Run Hardware Simulator

```
sh ./tools/HardwareSimulator.sh
```

### Run CPU Emulator

```
sh ./tools/CPUEmulator.sh
```

### Assemble

```
sh ./tools/Assembler.sh <path_to_asm>
```
